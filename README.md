# DoorDash — Multiplayer Board Game

A turn-based multiplayer board game built in Java for a university software engineering course.

## What it is
Players choose a monster hero and race across a board to reach the finish with enough energy to win. 
Each hero has unique powerups, the board has special cells, and a card system adds strategic depth.

## Key concepts applied
- OOP: abstract classes, inheritance, polymorphism, interfaces
- 4 monster types with unique overridden behaviors (Dasher, Dynamo, Schemer, MultiTasker)
- 5 card types with rarity-weighted draw system
- CSV-driven data loading — game config fully decoupled from logic
- Custom exception handling
- Java Swing GUI

## Built with
Java · Eclipse · Java Swing
